# recordable

> recordable histogram

## Usage

### Install

```bash
npm i @nicholaswmin/recordable
```

## Tests

Install deps

```bash
npm ci
```

Run unit tests

```bash
npm test
```

Run test coverage

```bash
npm run test:coverage
```
